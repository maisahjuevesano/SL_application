## Getting Started

### Run locally:

Clone the project

```terminal
  git clone https://github.com/maisahjuevesano/SL_application.git
```

Go to the project directory

```terminal
  cd SL_application
```

Install dependencies

```terminal
  npm install
```

Start the server

```terminal
  npm run dev
```
